# 软件开发工具实践课程展示网站

本项目是《软件开发工具实践》课程的小组实践项目，基于 Hexo 搭建，用于记录课程实验过程、工具使用方法和学习总结。

## 项目目标

通过本项目，完成从本地开发环境配置、Markdown 内容编写、Hexo 网站构建、Git 版本管理到 GitHub 远程仓库推送的完整实践流程。

## 技术栈

- Visual Studio Code
- Node.js
- npm
- Hexo
- Markdown
- Git
- GitHub
- SSH

## 本地运行方式

在项目根目录下执行以下命令：

    npm install
    hexo server

然后在浏览器中访问：

    http://localhost:4000

## 项目结构说明

- `_config.yml`：Hexo 网站总配置文件
- `source/_posts`：博客文章目录
- `themes`：网站主题目录
- `package.json`：项目依赖说明文件
- `node_modules`：npm 安装的项目依赖
- `public`：Hexo 生成的静态网站文件
